  Adverteerders → Iemand toevoegen.

2) INVUL-LINK FIXEN
De link die nu wordt gegenereerd werkt niet. Fix dit als volgt:

- Zorg dat de gegenereerde link:
  - een ABSOLUTE URL is
  - naar de juiste portal route wijst
- Gebruik voor de link:
  - `PUBLIC_PORTAL_URL` (env var)
  - fallback naar huidige host als deze env ontbreekt

Voorbeeld (concept):
`${PUBLIC_PORTAL_URL || currentOrigin}/portal/adverteerder/${token}`

- Controleer dat:
  - de frontend route `/portal/adverteerder/:token` bestaat
  - deze route GEEN admin-auth vereist
  - de token correct wordt doorgegeven aan de portal API

3) ROUTING CONTROLE
